## 🎯 现在只需 3 步

### 第 1 步：上传到 GitHub（5 分钟）

**最简单的方法 - 使用网页上传：**

1. **创建仓库**
   - 打开：https://github.com/new
   - 仓库名：`HabitBloom`
   - 选择 Public 或 Private
   - **不要**勾选任何初始化选项
   - 点击 "Create repository"

2. **上传文件**
   - 在仓库页面，点击 "uploading an existing file"
   - 将整个 `HabitBloom` 文件夹拖进去
   - 点击 "Commit changes"

**或者使用 GitHub Desktop：**
- 下载：https://desktop.github.com/
- 安装后：File → Add Local Repository → 选择 HabitBloom
- Publish repository → 选择仓库 → Publish

### 第 2 步：触发打包（1 分钟）

1. 在 GitHub 仓库页面，点击 **"Actions"** 标签
2. 点击左侧 **"Build HabitBloom APK"**
3. 点击右侧 **"Run workflow"** 按钮
4. 再次点击绿色的 **"Run workflow"** 按钮

### 第 3 步：等待并下载（10-20 分钟）

1. 等待打包完成（可以看到实时进度）
2. 打包完成后，点击运行记录
3. 滚动到底部，在 **"Artifacts"** 部分下载 APK

---

## 📱 安装到手机

1. 将 APK 文件传输到手机
2. 开启「允许安装未知来源应用」
3. 点击 APK 安装
