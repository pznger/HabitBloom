# 🌱 HabitBloom - 让习惯如花般绽放

一款基于可视化花园的个人习惯养成桌面应用，使用PyQt5开发。

## ✨ 核心特色

- 🌱 **可视化激励系统**：习惯进度以植物生长状态呈现
- 📊 **本地数据处理**：所有数据存储在设备本地，无需网络
- 🎯 **智能提醒系统**：基于用户行为的自适应提醒
- 🌈 **个性化花园**：可自定义植物种类、花园主题
- 📈 **数据统计**：本地生成习惯完成统计图表

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行应用

```bash
python main.py
```

## 🎮 功能模块

### 🏡 花园视图
- 展示所有习惯对应的植物
- 实时显示植物生长状态和健康度
- 一键浇灌（打卡）功能
- 按类别筛选

### 📋 习惯管理
- 添加/编辑/删除习惯
- 设置目标频率和难度等级
- 选择植物类型和图标
- 查看打卡历史

### 📊 统计分析
- 月度完成率统计
- 周统计图表
- 连续打卡排行榜
- 成就系统

### ⚙️ 设置
- 用户信息设置
- 深色模式切换
- 数据导入/导出
- 提醒时间设置

## 🌸 植物系统

植物生长阶段：
1. 🌱 种子 (0-20%)
2. 🌿 发芽 (20-40%)
3. 🪴 幼苗 (40-70%)
4. 🌸 开花 (70-100%)
5. 💐 结果 (100%)

植物类型：
- 🌸 花朵 (flower)
- 🌳 树木 (tree)
- 🌵 仙人掌 (cactus)
- 🌿 草药 (herb)

## 🏆 成就系统

- 🥉 初见坚持：连续打卡7天
- 🥈 习惯形成：连续打卡21天
- 🥇 习惯大师：连续打卡66天
- 🏆 传奇坚持者：连续打卡100天
- 💎 完美一周：一周内所有习惯全部完成
- 🌟 习惯收藏家：同时养成5个习惯
